# SPEC.md

## Project
商品撮影支援アプリの試作版

## Goal
複数枚の商品写真から、写り込みや反射の少ないベストな1枚を作る。
その後、ユーザーが1本のスライダーで「実物寄り ↔ 映え寄り」を微調整できる。

## MVP scope
今回作るものは以下のみ。

1. 画像を3〜5枚読み込める
2. 読み込んだ画像を一覧表示できる
3. 仮の合成処理を実行できる
4. 結果画面を表示できる
5. 1本のスライダーで見た目を変えられる
6. 保存できる

## Out of scope
今回は以下は作らない。

- 本格的なAIモデルの組み込み
- カメラ撮影ガイド
- ユーザー認証
- 課金
- クラウド同期
- 商品カテゴリ分類
- 自動出品連携

## UX policy
- 操作はできるだけ少なくする
- 難しい専門用語を出さない
- 詳細調整より、まず1本スライダーを優先する

## Definition of done
- 3枚以上の画像を選べる
- 結果画面まで遷移できる
- スライダー操作で画像表示が変わる
- 保存ボタンが動く
